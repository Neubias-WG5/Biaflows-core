BIAFLOWS

A Bio Image Analysis workflows benchmarking platform.

BIAFLOWS aims to help diffusing reproducible automated image analysis workflows and safeguard research based on their results by ensuring the highest performance standards.

How to use BIAFLOWS

  • Select BIA problem (Projects)
  • Explore image datasets held in this project (Explore > Images)
  • Select workflow to run on all image datasets (Explore > Jobs)
  • Set workflow functional parameters (dialog box, requires user account with run rights!)
  • Visually check results from past workflow runs (Explore > Images)
  • Explore benchmarking metrics results (Explore > Benchmarking)
